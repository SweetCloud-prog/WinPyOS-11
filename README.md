**WinPyOS 11** is an IT development project designed to replicate the visual experience and core functionalities of Windows 11 inside an isolated Python environment. It features a fully interactive desktop layout, native system apps, secure file sandboxing, and a modern UI rendering engine following Fluent design principles.

Thanks to my IT Teacher, M. Audras.

## ✨ Core Features

### 🖥️ Desktop Environment
- **Taskbar & Start Menu**: Centered taskbar alignment featuring an interactive start menu, operational search bar, and system shortcuts.
- **Window Management**: Complete window behavior mimicking native OS execution (minimize, maximize, close, and drag-and-drop movement).
- **Desktop Grid**: Dynamic desktop icon spawning with real-time application binding.

### 📂 Integrated Native Apps
- **File Explorer**: Dedicated storage navigation utility mapping the local `user_data` directory structure.
- **Notepad**: Built-in text editor capable of creating, reading, updating, and saving `.txt` files.
- **Media System**: Integrated rendering pipelines to open image files (`.jpg`) and play audio sequences (`.mp3`).
- **Terminal Companion**: An emulated shell layer designed to run internal Python testing sequences.

### 🛡️ Architecture & Security
- **Sandboxed User Data**: All user-generated files are strictly contained within a structural subfolder, protecting host system integrity.
- **Robust Error Catching**: Shielded runtime execution loops preventing global emulator crashes when handling corrupted inputs or faulty text file trees.
